API Documentation

Welcome to the Russell Marina Reservation Management API documentation. This API allows you to manage catways and reservations efficiently.

Tutorial

To use the API, you can make HTTP requests to the endpoints listed below. Ensure you have the necessary permissions for each action.

Catways

List All Catways

    GET /catways

Get a Specific Catway

    GET /catways/:id

Create a New Catway

    POST /catways
    {
        "name": "Catway 1",
        "location": "Paris"
    }

Response

    {
        "id": "1",
        "name": "Catway 1",
        "location": "Paris"
    }

Update a Specific Catway

    PUT /catways/:id
    {
        "name": "Updated Catway",
        "location": "Updated Location"
    }

Delete a Specific Catway

    DELETE /catways/:id

Reservations

List All Reservations for a Specific Catway

    GET /catways/:id/reservations

Create a New Reservation for a Specific Catway

    POST /catways/:id/reservations
    {
        "userId": "123",
        "date": "2024-10-01"
    }

Response

    {
        "id": "456",
        "catwayId": "1",
        "userId": "123",
        "date": "2024-10-01"
    }

Delete a Specific Reservation

    DELETE /catways/:id/reservations/:idReservation

Glossary

  • Catway: A type of boat dock.
  • Reservation: A booking for a specific catway.
