API Documentation

Welcome to the Russell Marina Reservation Management API documentation.

Catways

  • GET /catways - List all catways
  • GET /catways/:id - Get a specific catway
  • POST /catways - Create a new catway
  • PUT /catways/:id - Update a specific catway
  • PATCH /catways/:id - Partially update a specific catway
  • DELETE /catways/:id - Delete a specific catway

Reservations

  • GET /catways/:id/reservations - List all reservations for a specific catway
  • GET /catways/:id/reservations/:idReservation - Get a specific reservation
  • POST /catways/:id/reservations - Create a new reservation for a specific catway
  • DELETE /catways/:id/reservations/:idReservation - Delete a specific reservation
